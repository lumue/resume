# Resume

Use
- Replace `src/photo.jpg` with your photo.
- Update `src/resume.md`.
- Run `make.sh`

- Requirements
- [Python](https://www.python.org/)
- [Pandoc](https://pandoc.org/)
- pdflatex (included in [MiKTeX](https://miktex.org))
